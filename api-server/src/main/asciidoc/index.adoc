[[공통]]
== 공통사항

=== 공통 HTTP status codes

|===
|Status code |Usage

|`200 ok`
|성공

|`201 Created`
|등록성공

|`400 Bad Request`
|잘못된 요청

|`401 Unauthorized`
|권한 없음

|`404 Not Found`
|리소스를 찾을 수 없습니다.

|`405 Method Not Allowed`
|메서드 허용안 됨

|`412 Precondition Failed`
|검증 실패

|`415 Unsupported Media Type`
|Unsupported Media Type

|`500 Internal Server Error`
|시스템 에러
|===

=== 공통 Response Body
|===
| field | 설명

| `result`
| 응답 결과

| `reason`
| 실패 메시지

| `data`
| 실제 반환될 데이터

| `fieldErrors`
| 필드 에러 정보목록
|===
